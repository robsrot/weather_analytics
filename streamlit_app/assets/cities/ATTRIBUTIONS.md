## License compliance

- **CC0 / Public Domain**: no attribution required.
- **CC BY / CC BY-SA**: attribution required when displayed publicly; this file serves as the attribution record.
- Verify each image's license at the Wikimedia page URL before public deployment.
- Images resized to ≤1200px long edge, JPEG-80, via Pillow for performance.

## Re-fetching

Run `python scripts/fetch_city_images.py` to refresh images.
Set `FORCE=1` to re-download all (overwrites existing files).
